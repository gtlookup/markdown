# 概念

正向：用高级语言写一个可执行程序

- C/C++   ->   汇编   ->   机器语言

逆向：反过来，根据可执行程序反推出代码逻辑

- 机器语言   ->   汇编

动态反汇编：目标程序执行起来后的辅助程序，比如游戏外挂（修改不了磁盘上的数据，但可以修改内存中的数据）

静态反汇编：比如破解收费软件

# 工具

IDA：静态反汇编工具
