# 入门

```bash
# [] 字符串集，匹配[]中任何`一个`字符
[Rr]ust     # 匹配如： Rust、rust
[0-9a-zA-Z] # 匹配数字和字母。- 表示从哪到哪如0-9，
[\-]ust     # 匹配：-ust。\ 表示匹配特殊字符（需要转意）
[^0-9]ust   # 匹配如：aust、Aust等。^ 表示除了0-9都匹配
```

```bash
# 快捷方式：
# \d 匹配一个数字
# \D 匹配一个非数字
# \w 匹配一个数字或大小写字母或下划线
# \W 西区一个非数字非大小写字母非下划线的字符
# \s 匹配一个空白
# \S 匹配一个非空白
# \b 单词边界
1\sust  # 匹配：1 ust
\babc\b # 当 `aaa,abc-sdf-abc sdfds abc`，会匹配到3个abc
```

```bash
# 开始和结束：^ 写在[]外表示start_with，$ 表示end_with
^rust$ # 只匹配 `rust`，多任何字符或空格或换行都不匹配
rust$  # `abcrust`，匹配 `rust`
```

```bash
# 任意一个字符：. 匹配除了换行符(\n)
```

```bash
# 可先字符：? 表示?前的字符可以出现0次或1次
rus?t # 匹配 rust、rut
```

```bash
# 重复：
#   {n}    表示{n}前面的表达式出现n次
#   {m,n}  表示{m,n}前面的表达式出现m到n次
#   {m,n}? 贪婪模式，表示m次
#   {m,}   m次以上
\d{3}    # 匹配3个数字
\d{3,5}  # 匹配3、4、5个数字
\d{3,5}? # 匹配3个数字
\d{3,}   # 匹配3个以上数字
```

```bash
# 开闭区间：+ 等价于{1,}，* 等价于{0,}
f.+ # 匹配f开头的内容
```

# 进阶

```bash
# 分组：()内的表达式为一组
```

```bash
# `或`条件：|
```

```bash
# 非捕获分组：?:
```

```bash
# 分组的回溯引用：\n n表示第n个分组
abc<h1>标题</h1>123  # 想匹配
abc<h1>标题</div>123 # 不想匹配
<(.+)>(.+)</(\1)>   # \1表示重用第1个(.+)
<(.+)>(.+)</(\2)>   # 则匹配：<div>标题</标题>
```

```bash
# `环视`或`预搜索`或先行/后行断言，先行断言`从左往右`看，后行断言`从右往左`看
# 示例内容：我喜欢你 我喜欢 我喜欢他 喜欢 喜欢你

#   正向先行断言：(?=表达式) 指某个位置右侧必须匹配表达式
喜欢(?=你) # 只提取了`我喜欢你`和`喜欢你`中的`喜欢`

#   反向先行断言：(?!表达式) 作用是保证右边不能出现某个字符
喜欢(?!你) # 提取出`喜欢`但不能有`你`

#   正向后行断言（可能不支持所有浏览器）：(?<=表达式) 指某个位置左侧必须匹配表达式
(?<=我)喜欢(?=你) # 取出前面有`我`，后面有`你` 的 `喜欢`

#   反向后行断言（可能不支持所有浏览器）：(?<!表达式) 表示某个位置左侧不能匹配的表达式
(?<!我)喜欢(?!你) # 取出前面没有`我`、后面没有`你`的`喜欢`
```
